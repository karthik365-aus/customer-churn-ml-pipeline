# Customer Churn Prediction ML Pipeline

## Business Problem
Customer churn is a major challenge for banks and telecom companies. 
Predicting which customers are likely to leave enables targeted retention strategies.

## Dataset
Telco Customer Churn Dataset

Features include:
- Tenure
- Monthly Charges
- Contract Type
- Payment Method
- Internet Service

## Project Pipeline

1. Data Cleaning
2. Feature Engineering
3. Train/Test Split
4. Classification Models
   - Logistic Regression
   - Random Forest
   - Decision Tree
5. Model Evaluation

## Tools Used

- R
- tidyverse
- caret
- machine learning classification models

## Results

The models successfully identified patterns in customer churn behavior and provided interpretable predictors such as contract type and tenure.
